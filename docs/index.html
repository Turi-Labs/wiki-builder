What wiki-builder does

`wiki-builder` is a CLI that uses LLMs to build and maintain a personal knowledge base as a structured markdown wiki. You add source documents, and the tool writes summaries, concept pages, entity pages, cross-references, and a running index.

Why this approach is different

Traditional document tools often recompute context on every question. `wiki-builder` instead integrates source material into a persistent wiki so knowledge accumulates over time. Querying happens against that already-synthesized knowledge base.

Core workflow

  1. Initialize a wiki project.
  2. Add source documents into `raw/`.
  3. Run `wiki ingest` to let the LLM update the wiki.
  4. Use `wiki query` to ask questions across the accumulated pages.
  5. Run `wiki lint` and `wiki status` to keep the wiki healthy and inspectable.

Wiki layout

raw/                  # source documents you provide
wiki/                 # markdown knowledge base maintained by the LLM
schema.md             # behavioral rules and structural conventions
