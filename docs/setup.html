Installation

git clone https://github.com/yourname/wiki-builder
cd wiki-builder
pip install -e .

This installs the `wiki` command globally in your environment.

Dependencies

The package depends on `anthropic`, `openai`, `typer`, and `rich`.

API keys

Set an API key with environment variables:

# Anthropic
export ANTHROPIC_API_KEY=sk-ant-...

# OpenAI
export OPENAI_API_KEY=sk-...

Or save configuration with the CLI:

wiki config --provider anthropic --api-key sk-ant-...
wiki config --provider openai --api-key sk-...

Quickstart

wiki init my-research/ --name "AI Research"
cp ~/papers/attention-is-all-you-need.md my-research/raw/
cd my-research
wiki ingest raw/attention-is-all-you-need.md
wiki query "what is self-attention and why does it matter?"
wiki lint
wiki status

What `wiki init` creates

my-research/
├── schema.md
├── .gitignore
├── raw/
│   └── assets/
└── wiki/
    ├── index.md
    ├── log.md
    ├── concepts/
    ├── entities/
    └── sources/

Recommended working style

  • Open the wiki root in Obsidian or VS Code.
  • Ingest sources one at a time for better synthesis quality.
  • Edit `schema.md` when you want different page structure or conventions.
  • Keep `raw/` immutable so it remains your source of truth.
  • Use git in the wiki root so every ingest becomes a natural commit point.
