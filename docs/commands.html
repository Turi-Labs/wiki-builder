Reference for the `wiki` CLI.

These commands come directly from the current repository README and describe the main workflow for creating, growing, and checking a wiki.

`wiki init <path>`

Scaffolds a new wiki project.

wiki init my-research/
wiki init my-research/ --name "ML Paper Notes"

`wiki ingest <source>`

Reads a source document and integrates it into the wiki by writing a source summary, updating entity and concept pages, refreshing the index, and appending to the log. 

wiki ingest raw/paper.md
wiki ingest raw/article.md --provider openai --model gpt-4o
wiki ingest raw/notes.md --thinking
wiki ingest raw/paper.pdf --delete-pdf

Supported input formats are text-like files such as `.md`, `.txt`, `.rst`, and plain text. PDFs can be converted to sibling `.txt` files first when `pdftotext` is installed.

`wiki query "<question>"`

Asks a question against the existing wiki and synthesizes an answer from relevant pages.

wiki query "what is the transformer architecture?"
wiki query "compare attention mechanisms across these papers" --format table
wiki query "summarize everything I know about RLHF" --save
wiki query "what are the key open problems?" --output open-problems.md
Flag Description
--format markdown Default, structured markdown output.
--format table Returns the answer as a comparison table.
--format marp Creates Marp slides for tools like Obsidian with the Marp plugin.
--save Saves the answer back into the wiki as a new concept page.
--output <file> Writes the answer to a file instead of printing it.

`wiki lint`

Runs a health check across the wiki.

wiki lint
wiki lint --fix
  • Finds orphan pages.
  • Checks missing cross-references.
  • Flags contradictions and stale claims.
  • Detects concepts that should have their own pages.
  • Checks whether `index.md` is missing or outdated.

`wiki status`

Shows a local summary of the wiki without calling the LLM.

wiki status

`wiki config`

Manages provider, model, and API key settings in `~/.wiki-builder/config.json`.

wiki config --provider anthropic --api-key sk-ant-...
wiki config --provider openai --api-key sk-...
wiki config --model gpt-4o-mini
wiki config --show
