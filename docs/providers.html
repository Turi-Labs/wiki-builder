Providers

Anthropic and OpenAI are both supported.

You can set a default provider globally or override the provider and model per command.

On this page

Providers and models Installation and setup Command reference Architecture and schema

Supported providers

Provider Default model Notes
anthropic claude-opus-4-6 Default provider. Supports --thinking for extended reasoning.
openai gpt-4o Supports full tool use. The --thinking flag has no effect.

Resolution order

  1. --provider and --model CLI flags
  2. WIKI_PROVIDER, ANTHROPIC_API_KEY, and OPENAI_API_KEY environment variables
  3. Saved config in ~/.wiki-builder/config.json
  4. Default values: Anthropic and claude-opus-4-6

Examples

# Use OpenAI for one command
wiki ingest raw/paper.md --provider openai

# Use a cheaper model for quick queries
wiki query "what is X?" --provider openai --model gpt-4o-mini

# Switch the default provider
wiki config --provider openai

# Override per session
WIKI_PROVIDER=openai wiki ingest raw/paper.md

Practical note

The README notes that a typical ingest can touch 5 to 15 files and may involve 10 to 20 tool-calling turns. Costs depend on source length, wiki size, and model choice.
